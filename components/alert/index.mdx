## API

| 属性 | 说明     | 类型                                         | 默认值 |
| ---- | -------- | -------------------------------------------- | ------ |
| **message** | 警告消息 | **string \| reactNode** | -       |
| **kind** | 警告类型 | 'info'/'positive'/'negative'/'warning'非必填 | 'info' |
| **type** | 警告框类型 | 'default'/'outline'非必填 | 'default' |
| **showIcon** | 是否显示关闭按钮 | boolean 非必填 | false |
|**closable**|是否启用关闭，否者就禁用|boolean 非必填|true|
|**description**|警告描述|**string** 非必填|-|
|**onClose**|关闭时触发的回调函数|(e: MouseEvent) => void|-|
|**afterClose**|关闭动画结束后触发的回调函数|() => void|-|
